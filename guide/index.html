    # DruxtJS Breadcrumb

    The Druxt Breadcrumb module adds a Drupal (opens new window) Breadcrumbs Vue.js component to your Nuxt.js (opens new window) frontend.

    # Breadcrumb component

    The Breadcrumb component uses the Druxt Router (opens new window) data to calculate the current breadcrumbs and passes them through to the configured render component.

    See the DruxtBreadcrumb component API documentation.
